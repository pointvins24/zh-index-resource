# zh-index-resource

本仓库用于归档和发布多个独立的 HTML 页面。这些页面以静态资源的形式存放，不指向特定网站或域名，仅为通用型页面集合。

## 项目简介

zh-index-resource 是一个静态 HTML 页面归档仓库。仓库内包含若干独立页面，每个页面可单独访问，适用于展示、测试或参考用途。页面内容不涉及具体业务或商业推广，保持通用与中性。

## 目录说明

```
zh-index-resource/
├── index.html          # 仓库入口页面（可选）
├── pages/              # 存放各独立 HTML 页面
│   ├── example1.html
│   ├── example2.html
│   └── ...
├── assets/             # 公共资源文件（如 CSS、JS、图片等）
│   ├── css/
│   ├── js/
│   └── images/
└── README.md           # 本文件
```

- `pages/`：放置所有独立 HTML 页面，每个页面可独立运行。
- `assets/`：存放页面所需的公共资源，按类型分目录组织。

## 页面归档说明

- 每个页面均为完整的 HTML 文档，可直接通过浏览器打开。
- 页面之间无相互依赖关系，可单独使用。
- 资源引用尽量使用相对路径，便于本地预览或部署至静态托管服务。
- 页面内容遵循通用规范，不包含特定站点或服务信息。

## 维护说明

- 欢迎提交 Pull Request 添加或修改页面，请保持页面独立性与通用性。
- 新增页面请放入 `pages/` 目录，避免直接修改根目录文件。
- 资源文件请统一放入 `assets/` 对应子目录，避免散落。
- 请勿在页面中加入外部链接、跟踪脚本或商业内容。
- 如有问题或建议，请提交 Issue。

## 使用方式

1. 克隆或下载本仓库。
2. 直接双击 `pages/` 下的 HTML 文件，即可在浏览器中查看。
3. 也可将整个仓库部署至任意静态文件服务器，通过 URL 访问各页面。
